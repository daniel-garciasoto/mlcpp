\doxysubsection{Detailed Description}
Container class for machine learning datasets. 

Stores features (input data) and labels (output data) for supervised learning tasks. Provides utilities for loading, splitting, and normalizing data.

void mlcpp\+::\+Dataset\+::normalize (\begin{DoxyParamCaption}{}{}\end{DoxyParamCaption})}



Normalizes all features to the range \mbox{[}0, 1\mbox{]} using min-\/max scaling. 

For each feature column\+: normalized\+\_\+value = (value -\/ min) / (max -\/ min)

This modifies the dataset in-\/place. Features with constant values (range = 0) are left unchanged.

\begin{DoxyNote}{Note}
Use this when you need features in a bounded range \mbox{[}0, 1\mbox{]} 

This is sensitive to outliers 

Time complexity\+: O(n \texorpdfstring{$\ast$}{*} d) where n = samples, d = features
\end{DoxyNote}
