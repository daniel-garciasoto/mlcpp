Detailed Description

K-Nearest Neighbors (KNN) classifier for supervised learning.

A non-parametric, lazy learning algorithm that classifies samples based on the majority vote of their k nearest neighbors in the feature space.

Note
This is a "lazy" algorithm - training only stores the data, all computation happens during prediction.
