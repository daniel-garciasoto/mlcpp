Constructs a KNN classifier with specified parameters.

Constructs a KNN classifier with specified parameters.

Parameters
kNumber of nearest neighbors to consider (default: 3)
distanceDistance metric function to use (default: euclidean_distance)
Note
k should be odd to avoid ties in binary classification
Larger k values make the model more robust but less sensitive to local patterns

KNN model1(5); // k=5, Euclidean distance KNN model2(3, manhattan_distance); // k=3, Manhattan distance
