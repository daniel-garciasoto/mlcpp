Detailed Description

Container class for machine learning datasets.

Stores features (input data) and labels (output data) for supervised learning tasks. Provides utilities for loading, splitting, and normalizing data.

Loads a dataset from a CSV file.

Supports both numeric and text labels (text labels are automatically mapped to integers). The CSV file should have features as columns and samples as rows.

Parameters
filepathPath to the CSV file
has_headerWhether the first row contains column headers (default: true)
label_columnIndex of the label column, -1 for last column (default: -1)
Returns
Optional Dataset object. Returns empty optional if loading fails.
Note
Text labels are automatically converted to numeric IDs (0, 1, 2, ...)
All feature columns must contain numeric values

Example usage:

auto dataset = Dataset::from_csv("data/iris.csv", true, -1);
if (dataset) {
cout << "Loaded " << dataset->size() << " samples" << endl;
}

◆ normalize()

void mlcpp::Dataset::normalize ( )

Normalizes all features to the range [0, 1] using min-max scaling.

For each feature column: normalized_value = (value - min) / (max - min)

This modifies the dataset in-place. Features with constant values (range = 0) are left unchanged.

Note
Use this when you need features in a bounded range [0, 1]
This is sensitive to outliers
Time complexity: O(n * d) where n = samples, d = features
See also
standardize() for an alternative normalization method

Example usage:

dataset.normalize();
// All features now in [0, 1] range

◆ standardize()

void mlcpp::Dataset::standardize ( )

Standardizes all features to have mean=0 and standard deviation=1.

For each feature column: standardized_value = (value - mean) / std_dev

This modifies the dataset in-place. Features with zero standard deviation are left unchanged.

Note
Use this when features have different scales and you care about distribution
Less sensitive to outliers than normalize()
Time complexity: O(n * d) where n = samples, d = features
See also
normalize() for min-max scaling alternative

Example usage:

dataset.standardize();
// All features now have mean=0, std=1

pair< Dataset, Dataset > mlcpp::Dataset::train_test_split ( double test_ratio = 0.1,
int seed = 41 ) const

Splits the dataset into training and testing sets.

Randomly shuffles the data before splitting to ensure random distribution. Uses a seed for reproducibility.

Parameters
test_ratioProportion of data to use for testing (between 0.0 and 1.0, default: 0.1)
seedRandom seed for reproducibility (default: 41)
Returns
Pair of datasets: {train_dataset, test_dataset}
Exceptions
std::invalid_argumentIf test_ratio is not between 0.0 and 1.0
Note
Time complexity: O(n) where n is the number of samples

Example usage:

auto [train, test] = dataset.train_test_split(0.2, 42); // 80% train, 20% test
cout << "Train: " << train.size() << " samples" << endl;
cout << "Test: " << test.size() << " samples" << endl;
